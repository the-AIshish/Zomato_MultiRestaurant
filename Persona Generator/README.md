# UI/UX Designer Persona Generator

A static web tool that turns a PRD or project brief into **3 distinct UI/UX designer personas** for AI coding tools such as **Cursor, Lovable, Replit, Windsurf, and v0**.

It is designed for founders, product managers, indie hackers, and developers who want better design direction before generating an app or website with AI.

---

## What this tool does

Paste in:
- a PRD,
- a rough startup idea,
- or a simple product description.

The tool will generate:
- **Designer 1:** Clean, trustworthy, structured
- **Designer 2:** Bold, expressive, memorable
- **Designer 3:** Futuristic, premium, hi-tech

Each persona includes:
- Name
- Design philosophy
- Visual preview
- Best use case
- Signature moves
- A copy-ready design prompt

---

## Why this is useful

AI coding tools often generate decent structure but weak design direction.

This tool helps you:
- choose a better visual direction faster,
- avoid generic AI-looking UIs,
- create stronger prompts for AI builders,
- keep design systems more consistent.

## How to upload to GitHub as a non-techie

If you don’t know coding or terminal commands, follow this exact method.

### Step 1: Create a GitHub account

1. Go to https://github.com
2. Sign up if you don’t already have an account
3. Verify your email

### Step 2: Create a new repository

1. After logging in, click the **+** icon on the top-right
2. Click **New repository**
3. Repository name: `uiux-persona-generator`
4. Keep it **Public** if you want GitHub Pages hosting
5. Click **Create repository**

### Step 3: Upload the files

1. Open your new repository
2. Click **Add file**
3. Click **Upload files**
4. Drag and drop these files:
   - `uiux-persona-generator.html`
   - `README.md`
5. In the message box, type:
   - `Initial upload`
6. Click **Commit changes**

### Step 4: Rename the HTML file if you want GitHub Pages

GitHub Pages works best when your homepage file is named `index.html`.

So do this:

1. In your repository, click `uiux-persona-generator.html`
2. Click the pencil/edit icon
3. Change the filename at the top from:
   - `uiux-persona-generator.html`
   to
   - `index.html`
4. Scroll down
5. Click **Commit changes**

### Step 5: Turn on GitHub Pages

1. In the repo, click **Settings**
2. Click **Pages** in the left menu
3. Under **Build and deployment**:
   - Source = **Deploy from a branch**
4. Branch = **main**
5. Folder = **/root**
6. Click **Save**

After a minute or two, GitHub will give you a public link.

It will look something like:

`https://yourusername.github.io/uiux-persona-generator/`
